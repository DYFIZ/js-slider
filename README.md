# js plugin which control photo gallery. You just need to add instance of slider class: 
```
new Slider({
     images: '.gallery .photo img',
     next: '.gallery .tabs .next',
     prev: '.gallery .tabs .prev',
     navigationImg: '#navigation div',
     interval: '2000',
     autoplay: true,
     navigation: true
   });
```
